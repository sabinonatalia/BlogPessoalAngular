<h2>Funcionalidades da aplicação</h2>
                                                                                                                      
- Cadastro de usuários;
- Login dos usuários;
- Criação de postagem;
- Editar/apagar postagem
- Cadastro de tema;
- Editar/ apagar tema;
- Pesquisa por título de postagem;
- Pesquisa por tema;
- Atualizar usuários;
